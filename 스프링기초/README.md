# 스프링 기초

간단한 웹 어플리케이션을 개발하면서 각 기술들이 어떻게 사용되는지 큰 틀 잡음

## 간단한 웹 어플리케이션 개발

* 스프링 프로젝트 생성
* 스프링 부트로 웹 서버 실행
* 회원 도메인 개발
* 웹 MVC 개발
* DB 연동 - JDBC, JPA, 스프링 데이터 JPA
* 테스트 케이스 작성

## 프로젝트 사용 기술

* spring boot
* Gradle
* Thymeleaf 
* HIBERNATE
* JPA

## 스프링 강의 진행 로드맵
* 스프링 입문
* 스프링 핵심 원리
* 스프링 웹 MVC
* 스프링 DB 데이터 접근 기술
* 실전! 스프링
